## 📝 Project Description

### Short Description (1 sentence)
> Evo-Cortex is an OpenClaw plugin that gives AI agents persistent memory, continuous learning, and evolutionary capabilities.

### Medium Description (1 paragraph)
> Evo-Cortex transforms AI agents from passive responders into active learners by providing a complete brain-like system. It features persistent memory with semantic search, automatic knowledge graph construction, experience distillation into meta-rules, and scheduled evolution cycles. Built for OpenClaw, it reduces costs by 78% and improves response times by 98% through innovative hybrid execution.

### Long Description (for press releases)
> Evo-Cortex v1.0.0 represents a breakthrough in AI agent architecture, introducing the first complete memory, learning, and evolution system for OpenClaw agents. Developed by the Evo-Agents team and published on npm as @evo-agents/evo-cortex, this plugin enables agents to remember past interactions, learn from patterns, and continuously improve over time.
>
> Key innovations include a SQLite-based semantic memory index, automatic knowledge graph construction, meta-rule generation from repeated experiences, and a sophisticated cron-based evolution scheduler with nine specialized tasks. The system achieves 78% cost reduction and 98% speed improvement through its hybrid Script+LLM execution model.
>
> "We wanted to solve the goldfish memory problem of AI agents," says the development team. "With Evo-Cortex, agents don't just respond—they accumulate wisdom."
